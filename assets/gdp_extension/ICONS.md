# Icon Files Required

Extension cần các file icon sau:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

## Cách tạo icons

Bạn có thể:
1. Sử dụng favicon.png từ thư mục `assets/` và resize thành các kích thước trên
2. Tạo icons mới với logo của bạn
3. Sử dụng công cụ online như https://www.favicon-generator.org/

## Tạm thời (để test)

Nếu chưa có icons, bạn có thể:
1. Tạo các file PNG đơn giản với màu nền (ví dụ: màu xanh #4a90e2)
2. Hoặc comment out phần icons trong manifest.json để test trước

Extension vẫn hoạt động bình thường dù không có icons, chỉ là sẽ không hiển thị icon trên toolbar.
